1 # Fast Event System [![Build Status](https://travis-ci.org/makiolo/fast-event-system.svg?branch=master)](https://travis-ci.org/makiolo/fast-event-system)
2 
3 This fast event system allows calls between two interfaces decoupled (sync or async)

5 ## Design guidelines in fast-event-system
6 
7 * Can assume variadic templates supported by your compiler.
8 * Use perfect forwarding and new features from C++11/14 when possible.
9 * Prefer header only code, but it is not a must.
10 
11 ## Quality assurance
12 
13 * Code tested in travis on gcc (4.7, 4.8, 4.9), clang (3.3, 3.4 and 3.6) and Visual Studio (2013).
14 * Test cases relationated with problems crossing boundaries of dynamic libraries.
